================================================================================

ISSUE REPORTED BY USER:
-----------------------
Question 20 in exam results showed:
  ❌ Red X mark at top (indicating wrong/missed)
  ✅ Green checkmark on "chew" (correct answer)

User thought this was contradictory and confusing.

================================================================================

ROOT CAUSE:
-----------
The display was TECHNICALLY CORRECT but VISUALLY CONFUSING:

What Actually Happened:
  1. Student SKIPPED Question 20 (visited but didn't select any answer)
  2. System correctly marked it as INCORRECT (0 marks) → Red X
  3. System correctly showed CORRECT ANSWER in green → "chew"
  4. BUT there was NO CLEAR INDICATION that student didn't answer

Result: It looked like student selected "chew" and got it right, but the 
        red X said it was wrong. Very confusing! 😕

================================================================================

SOLUTION IMPLEMENTED:
---------------------
Enhanced the result display to clearly show when questions are not answered.

1. ADDED "NOT ANSWERED" BANNER
   ⚠️ Yellow banner appears above options:
   "Not Answered - This question was skipped"

2. ENHANCED OPTION LABELS
   ✅ chew (Correct Answer)  ← Clear label
   ❌ wrong (Your Answer)     ← Clear label

3. VISUAL IMPROVEMENTS
   - Correct answer: Light green background
   - Wrong answer: Light red background
   - Better contrast and clarity

THREE QUESTION SCENARIOS:
-------------------------

1. ANSWERED CORRECTLY:
   - Green checkmark at top ✅
   - Selected option has green background + "(Your Answer)" label
   - No "Not Answered" banner

2. ANSWERED INCORRECTLY:
   - Red X at top ❌
   - Wrong option has red background + "(Your Answer)" label
   - Correct option has green background + "(Correct Answer)" label
   - No "Not Answered" banner

3. NOT ANSWERED (SKIPPED):
   - Red X at top ❌
   - Yellow "Not Answered" banner
   - Only correct option has green background + "(Correct Answer)" label
   - No student answer indicator
